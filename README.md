# 📊 Inventory Management Dashboard (Power BI)

## 📌 Project Overview
This project is a Power BI Inventory Management Dashboard designed to analyze and monitor stock levels, inventory value, and low stock alerts. It helps businesses track inventory performance and make data-driven decisions for procurement and stock control.

---

## 🛠 Tools & Technologies
- Power BI
- DAX (Data Analysis Expressions)
- Data Modeling
- Excel / SQL (Data Source)

---

## 📊 Key Features
- Total Inventory Value analysis
- Total Stock Quantity monitoring
- Low Stock detection system (alert-based KPI)
- Product-level analysis
- Category-based inventory breakdown
- Interactive filtering by product, category, supplier, and warehouse

---

## 🚨 Business Problem Solved
Businesses often face issues such as stock shortages, overstocking, and lack of visibility into inventory performance. This dashboard solves these problems by providing real-time insights into inventory levels and identifying critical low-stock items.

---

## 📈 Key Insights
- Identifies products that fall below reorder level
- Shows high-value inventory concentration by category
- Highlights overall stock availability across the business
- Helps optimize procurement and reduce stockouts

---

## 📌 Dashboard Highlights
- KPI Cards for quick business overview
- Conditional formatting for low stock alerts
- Category and product-level analysis
- Interactive and user-friendly design

---

## 🎯 Outcome
This dashboard enables better inventory control, improves decision-making efficiency, and supports operational planning by providing clear visibility into stock performance.
